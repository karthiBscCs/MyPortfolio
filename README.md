# Karthikeyan N — Portfolio

Personal portfolio website for **Karthikeyan N**, Senior .NET & Backend Engineer.

**Tech stack:** Pure HTML, CSS & Vanilla JS — no build step required.

## 🚀 Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Vercel auto-detects static site — click **Deploy**
5. Done! Your portfolio is live.

## 📁 Project Structure

```
portfolio/
├── index.html        # Main page
├── css/
│   └── style.css     # All styles
├── js/
│   └── main.js       # Interactions & animations
├── vercel.json       # Vercel deployment config
└── README.md
```

## ✏️ Customization

- **Fiverr link**: Search for `https://www.fiverr.com` in `index.html` and replace with your actual Fiverr profile URL
- **Service prices**: Update in the Services section of `index.html`
- **Colors**: Edit CSS variables in `:root` inside `css/style.css`
